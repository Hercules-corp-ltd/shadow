# Greek Mythology Themed Commits

All commits should use Greek mythology terms that fit the context.

## Commit Types with Greek Terms

- `feat:` - New features (use: Hermes, Apollo, Athena, etc.)
- `fix:` - Bug fixes (use: Hephaestus forge, Ares battle, etc.)
- `docs:` - Documentation (use: Oracle, Scribe, etc.)
- `style:` - Styling (use: Apollo's light, Aphrodite beauty, etc.)
- `refactor:` - Refactoring (use: Metamorphosis, Rebirth, etc.)
- `test:` - Tests (use: Trial, Challenge, etc.)
- `chore:` - Maintenance (use: Tending, Care, etc.)
- `perf:` - Performance (use: Hermes speed, etc.)

## Examples

```bash
git commit -m "feat: Add Hermes messenger for wallet delivery"
git commit -m "fix: Hephaestus forge repair in backend"
git commit -m "docs: Oracle documentation update"
git commit -m "style: Apollo's light theme improvements"
git commit -m "refactor: Metamorphosis of wallet provider"
git commit -m "feat: Zeus thunder for on-chain programs"
git commit -m "chore: Tending to Olympus gates"
git commit -m "perf: Hermes speed optimization"
git commit -m "feat: Hades shadow identity system"
git commit -m "fix: Atlas shoulders storage fix"
```

## Project Structure (Greek Gods)

- **Zeus** (Ζεύς) - On-chain programs (king of gods, thunder)
- **Hephaestus** (Ήφαιστος) - Backend server (god of forge, craftsmanship)
- **Apollo** (Απόλλων) - Frontend (god of light, prophecy)
- **Olympus** (Όλυμπος) - Standalone app (home of gods)
- **Hermes** (Ερμής) - SDK (messenger god, speed)
- **Hades** (Άδης) - Wallet/Identity (god of underworld, shadows)
- **Atlas** (Άτλας) - Storage (titan who holds the world)
